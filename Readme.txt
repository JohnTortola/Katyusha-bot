Needs:
-Python 3.x
-discord and youtube download libraries (py -m pip install discord yt-dlp)
-.env file library (py -m pip install python-dotenv)
-FFmpeg 
    	-Set FFmpeg Path: Make sure that the bin folder inside your FFmpeg directory (where ffmpeg.exe is located) is added to your system's PATH.
-Remember to have your bot token, audio, ffmpeg and ffprobe in a .env file
